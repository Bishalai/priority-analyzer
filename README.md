## Usage

```bash
    git clone https://github.com/ashishlamsal/priority-analyzer.git
    cd priority-analyzer/
    python -m venv venv
    ./venv/Scripts/activate
    pip install -r requirements.txt
    cd admission/
    python manage.py runserver

```
